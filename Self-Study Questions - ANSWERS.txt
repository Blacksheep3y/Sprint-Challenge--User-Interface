1. If you were to describe semantic HTML to the next cohort of students, what would you say?

    A: semantic html is a way of describing an html page to other web developers by way of it's 
    markup, it is a way for a page's information to be organized meaningfully, rather than stylistically or for presentation use.  

2. Name two big differences between display: block; and display: inline;.

    A1: Display block takes up the full width of the page and display inline sorts of push to remain inline with the other elements 
    so as long as there is space to do so.
    A2: Display block also takes in properties such as width and height, whereas display inline does not accept these properties.

3. What are the 4 areas of the box model?

    A: The 4 areas of the box model include: the content, the padding, the border, and the margin.

4. While using flexbox, what axis does the following property work on: align-items: center?

    A: This property works on the cross axis which by default is the vertical axis.

5. Explain why git is valuable to a team of developers.

    A: Git is valuable to a team of developers because Git is used to monitor and control code changes made during development across several people or teams,
    it is a way for developers to be able to go back to their code at any time in which it was altered/saved
    so that they can always back track through the work. This also allows for multiple developers to work on one project all at the same time.
